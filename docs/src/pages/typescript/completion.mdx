# Completion

The `Completion` endpoint allows you to give a prompt to an Anthropic LLM and get a respnose.

```typescript
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300
)
```

## Usage

```typescript
import AnthropicBedrock from "anthropic-bedrock";

const anthropic = new AnthropicBedrock({
  access_key: process.env["AWS_ACCESS_KEY"],
  secret_key: process.env["AWS_SECRET_KEY"],
});

async function main() {
    const completion = await anthropic.Completion.crearte(
        model: "anthropic.claude-v2",
        prompt: "In one sentence, what is good about the color blue?",
        max_tokens_to_sample: 300
    )

    console.log(completion.completion)
}

main();
```

## Configuration

### model

The model that will complete your prompt.

```typescript {2}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300
)
```

### prompt

The prompt you want to use.

```typescript {3}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300
)
```

### max_tokens_to_sample

The maximum number of tokens to generate before stopping.

```typescript {4}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300
)
```

### stop_sequences (optional)

Sequences that will cause the model to stop generating completion text.

```typescript {5-7}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300,
    stop_sequences:[
        "sequence"
    ]
)
```

### temperature (optional)

Amount of randomness injected in the response. 0-1

_Note: you can either use temperature, top p, or top k._

```typescript {5}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300,
    temperature: 0.7
)
```

### top_p (optional)

Use nucleus sampling. 0-1

_Note: you can either use temperature, top p, or top k._

```typescript {5}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300,
    top_p: 0.7
)
```

### top_k (optional)

Only sample from the top K options for each subsequent token. 0-1

_Note: you can either use temperature, top p, or top k._

```typescript {5}
const completion = await anthropic.Completion.crearte(
    model: "anthropic.claude-v2",
    prompt: "In one sentence, what is good about the color blue?",
    max_tokens_to_sample: 300,
    top_k: 0.7
)
```
